# Diaspora (Markdown → Dataset para LLMs)

## Uso rápido
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r diaspora/requirements.txt
python diaspora/main.py -i diaspora/data/input -c diaspora/config/settings.yaml -o diaspora/data/output -p diaspora/data/processed --log-level INFO
```
